IoT Project

WiFi-based ESP32 project for real-time LDR sensor monitoring and servo motor control through a web interface.

Project Description

In this project, I used the ESP32 to create a WiFi-based system that interacts with both an LDR sensor and a servo motor. The LDR sensor reads light intensity values, while the servo motor angle can be adjusted via a web interface. Through a simple web server hosted on the ESP32, the user can toggle a LED, read sensor data and control the servo's position in real-time.

This project demonstrates a practical application of IoT principles, showcasing how embedded systems can collect data and interact with the physical world through remote web interfaces. It is a valuable project as it highlights my ability to integrate hardware (sensors, actuators) with software (WiFi, web server) using HTML, CSS, JavaScript, and Arduino programming.
