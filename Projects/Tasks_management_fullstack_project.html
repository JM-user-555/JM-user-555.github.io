SmartTask: Full-Stack Personal Task Manager

A Modern React, Node.js & MySQL Application for Secure, Personalized To-Do Management

Project Description

This full-stack web project is a modern task management application featuring user authentication, personalized dashboards, and dynamic task handling. Built with a React frontend and an Express/Node.js backend connected to a MySQL database, it allows users to sign up, log in securely, and manage their own to-do lists. Each user has a dedicated profile page where they can add and remove tasks, with all data operations performed in real time and persisted in the database. The application also includes a responsive, visually appealing UI with light/dark theme support, demonstrating both frontend and backend integration, RESTful API design, state management, and secure user data handling.

Technical Overview

  • User Authentication: Secure signup and login using hashed passwords with bcrypt.
  • Personalized Dashboards: Each user has a unique profile page displaying their information and tasks.
  • Task Management: Users can add, view, and delete tasks, with all operations reflected in the MySQL database.
  • RESTful API: Backend exposes RESTful endpoints for user and task operations.
  • React Frontend: Modern, responsive UI built with React, including state management with hooks.
  • Express/Node.js Backend: Handles API requests, authentication, and database communication.
  • MySQL Database: Stores user credentials and tasks with relational integrity.
  • Light/Dark Theme Toggle: Users can switch between light and dark modes for better UX.
