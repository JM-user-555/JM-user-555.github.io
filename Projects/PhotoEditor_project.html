Project Description

This project is a photo editor website built using HTML, CSS, and JavaScript, allowing users to upload and manipulate images with various visual properties. The editor provides functionalities such as adjusting the border style, color, and size, modifying border radius, altering opacity, applying grayscale filters, rotating the image, and resizing it.

Key features include image manipulation, where users can customize images by changing borders, opacity, grayscale, and rotation settings. and JavaScript integration through complex interactions like real-time image adjustments.

This project highlights practical skills in front-end web development, showcasing the ability to integrate HTML, CSS, and JavaScript to create interactive and user-friendly interfaces. It demonstrates proficiency in manipulating DOM elements, handling user input, and implementing dynamic functionality. As a portfolio piece, it illustrates both creativity and technical expertise in building functional web applications.
