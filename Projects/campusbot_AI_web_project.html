Project Description

This project is a web-based chatbot application called CampusBot, designed to assist university students and visitors by answering questions related to academic, financial, and campus life topics. Built using Python, Flask, HTML, CSS and JavaScript, the chatbot leverages string matching to understand user queries and provide relevant responses from a curated FAQ or frequently asked questions. The user interface features a modern, responsive design with real-time chat functionality, making it easy for users to interact and receive instant support about university information, admission processes, faculty details, and more. This project demonstrates skills in web development, natural language processing, and user experience design.

Technical Overview

  • Immediate AI Responses: Uses fuzzy string matching to understand and match user questions to a curated FAQ on the backend side and provides information on academics, faculties, financial aid, admissions, contact details, and more.
  • Interactive Real-Time Chat Web Interphase: Interactive web-based chatbot for university-related queries with a modern, responsive UI design and clear separation of user and bot messages.
  • Python Backend using Flask: Built with Python (Flask) for the backend, handling requests from users as questions and responses from the bot as answers.
  • JavaScript Frontend: Utilizes JavaScript for dynamic content updates and real-time chat functionality, enhancing user interaction.
  • Responsive Design: Ensures compatibility across devices, providing a seamless user experience on both desktop and mobile platforms.
