Programmed an agent that was able to read and solve Sudoku.

Project Description ML Sudoku Solver

In this code, after building an AI agent that solves the sudoku using csp and backtracking, we are going to make a model that read the Sudoku as image and solve the Sudoku and print the correct numbers on the image.
Main Rule: Each row, column and block (9 spaces each) needs to be filled out with the numbers from 1 to 9, without repeating any numbers within the row, column or square. The solution should be returned based on the rule indicated above, no matter whether the sudoku is in easy or medium or hard mode.
