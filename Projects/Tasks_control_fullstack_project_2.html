TaskFlow: Modern Full-Stack Task Management Platform

A robust web application for seamless task assignment, tracking, and real-time notifications with secure, role-based access for managers and employees.

Project Description

This full-stack web project is a Task Management System designed for organizations to efficiently assign, track, and manage tasks between managers and employees. Built with a modern React frontend and a Node.js/Express backend connected to a relational database, the application supports user authentication, role-based access (manager vs. employee), task assignment with due dates, real-time notifications, and a clean, responsive user interface. Managers can assign tasks to employees, monitor progress, and receive notifications when tasks are completed, while employees can view, complete, and manage their assigned tasks.

Technical Overview

  • Role-Based Authentication: Secure login system with distinct access and features for managers and employees with hashed passwords.
  • Task Assignment & Tracking: Managers can assign tasks to employees, set due dates, and track completion status.
  • Due Date Management: Each task includes a due date and time, with overdue tasks visually highlighted with all operations reflected in the MySQL database.
  • Real-Time Notifications: Users receive notifications for important events, such as task completion or assignment.
  • React Frontend: Modern, responsive UI built with React, including state management with hooks.
  • Express/Node.js Backend: Handles API requests, authentication, and database communication.
  • MySQL Database: Stores user credentials and tasks with relational integrity.
  • Light/Dark Theme Toggle: Users can switch between light and dark modes for better UX.
